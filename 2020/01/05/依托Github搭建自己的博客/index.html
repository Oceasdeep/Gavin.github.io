依托Github搭建自己的博客

Posted on 2020-01-05

欢迎来到 我的博客! 很长时间以来就想着写自己的博客了，现在终于下手了[手动搞笑]，第一篇博客就介绍一下怎么依托Github搭建自己的博客，希望和大家交流学习，也是督促自己不断学习，努力成长为对自己、公司、社会和国家有一点点恭喜的人[手动鼓掌]。

话不多说，我下面结合自己的实践把这第一篇博客写好，希望对于你有一点借鉴，如果觉得写得还可以话，就给个star鼓励一下吧。

环境

win10 64位

所需的软件

Git、Node.js、Hexo

软件安装

Git安装

对于Windows系统而言直接下载Git的软件安装包安装即可。Git官网,如果网络下载不给力可以使用这个Git-2.23备用链接，永久有效提取码：otib

说明：安装完Git查看是否安装成功，在cmd或者GIt bash中输入以下命令：

git --version

如果出现此信息说明安装成功! cmd中查询结果

Node.js安装

对于Windows系统而言直接下载Node的软件安装包安装即可。 Node官网,如果网络下载不给力可以使用这个Node-12.14备用链接，永久有效提取码：vptz

说明：Node.js后输入以下命令检查安装是否成功

node -v        //检查安装Node.js
npm -v        //用于安装Hexo
